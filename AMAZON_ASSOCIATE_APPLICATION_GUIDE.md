## 🚀 Amazon Associate申請手順

### 1. Associate管理画面にログイン
```
URL: https://affiliate.amazon.co.jp
ID: (既存のアカウント)
```

### 2. 新サイトの追加申請
1. **「アカウント情報」** メニューをクリック
2. **「Webサイト情報の管理」** を選択
3. **「Webサイト追加」** ボタンをクリック

### 3. サイト情報の入力
```
サイト名: 盆栽カタログ
URL: https://your-deployed-site-url.netlify.app
説明: 盆栽愛好家向けの商品紹介・比較サイト。初心者から上級者まで、適切な盆栽選びをサポートします。

主な内容:
- 盆栽樹木の詳細情報と比較
- 盆栽用品・道具の紹介
- 価格比較とレビュー
- 初心者向け育成ガイド

対象商品カテゴリ:
- 園芸・ガーデニング
- ペット・ペット用品（盆栽関連）
- DIY・工具・ガーデン
```
